Уровни доступа

В системе предусмотрены следующие уровни прав доступа:

Администратор

Полный доступ ко всем функциям системы: управление пользователями, журнал действий, все разделы

Расширенный доступ

Доступ к панели управления, создание заданий, управление заказами, сотрудниками и работами

Стандартный доступ

Создание и просмотр заданий, доступ к отчетам

Ограниченный доступ

Только просмотр заданий без возможности создания или редактирования

Работа с заданиями

Создание задания

  1. Нажмите кнопку «Новое задание» в шапке сайта
  2. Заполните необходимые поля: сотрудник, вид работы, заказ, количество часов
  3. Нажмите кнопку «Создать задание»
  4. После успешного создания появится уведомление, и вы будете перенаправлены на страницу заданий

Фильтрация заданий

На странице заданий доступны следующие фильтры:

  • Подразделение — выбор одного или нескольких подразделений
  • Дата — диапазон дат для выборки
  • ФИО сотрудника — поиск по имени
  • Номер заказа — поиск по номеру
  • Наименование работы — поиск по названию
  • Наименование заказа — поиск по названию

Действия с заданиями

Редактирование

Нажмите на иконку карандаша в строке задания

Удаление

Нажмите на иконку корзины, подтвердите действие в модальном окне

Экспорт

Нажмите кнопку «Выгрузить отчет» для сохранения данных

Подсчет часов

Кликните на ячейку с часами, чтобы выделить задание. Внизу страницы отобразится сумма выбранных часов. Кнопка сброса позволяет очистить выборку.

Управление заказами

Создание заказа

  1. Перейдите в раздел «Заказы» панели управления
  2. Нажмите кнопку «Добавить заказ»
  3. Заполните поля: номер заказа, название, описание (необязательно)
  4. Сохраните заказ

Редактирование и удаление

Для редактирования или удаления заказа используйте соответствующие иконки в таблице. При удалении потребуется подтверждение.

При удалении заказа связанные задания не удаляются, но становятся несвязанными.

Управление сотрудниками

Добавление сотрудника

  1. Перейдите в раздел «Сотрудники»
  2. Нажмите «Добавить сотрудника»
  3. Заполните данные: ФИО, табельный номер, подразделение, роль (рабочий/специалист/руководитель)
  4. Сохраните сотрудника

Роли сотрудников

  • Рабочий — выполнение производственных задач
  • Специалист — выполнение специализированных работ
  • Руководитель — управление и координация

Управление работами

В справочнике работ хранятся все доступные виды работ, которые можно назначать сотрудникам.

Добавление работы

  1. Перейдите в раздел «Работы»
  2. Нажмите «Добавить работу»
  3. Укажите наименование работы
  4. Сохраните

Формирование отчетов

Система позволяет экспортировать данные в формате Excel для дальнейшей обработки.

Способ формирования

  1. На странице заданий или в разделе «Отчеты» нажмите «Выгрузить отчет»
  2. Примените необходимые фильтры (по дате, сотруднику, заказу)
  3. Нажмите кнопку экспорта
  4. Файл будет загружен на ваш компьютер

Формат файла

Отчет формируется в формате .xlsx (Microsoft Excel) с табличным представлением данных и заголовками столбцов.
